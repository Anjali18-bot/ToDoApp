# To-Do List App (Python)

This is a simple command-line To-Do List application built using Python.

## Features

- Add tasks
- View all tasks
- Exit anytime

## How It Works

The program shows you 3 options:
1. Add a new task
2. View all your tasks
3. Exit the app

## Run the Project

You can run this in your terminal:

```bash
python3 todo.py
